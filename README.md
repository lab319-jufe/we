+ DataStrc R和Python基本的语法与数据结构
+ Gen-Stat 统计学的一般代码，包括描述统计、线性回归、非线性回归和方差分析
+ Non-Stru 非结构化数据，包括文本、图片、人脸检测
+ SEM 结构方程模型，包括EFA、CFA等
+ Stat-Learning 统计学习相关的内容，包括分类、决策树、回归分析（正则化）、交叉验证等
+ TS 时间序列分析
+ Visu 数据可视化，包括Python的Matplotlib等
